    API Calling & DOM in JavaScript – Complete Beginner to Interview Guide

    What is an API?

    Definition:
    API (Application Programming Interface) is an endpoint that provides data without exposing internal logic.

    From the session:

    "API is an endpoint. You don’t know what happens behind the scene — you just get data." :contentReference[oaicite:0]{index=0}

    ---

    Simple Real-Life Analogy

    • TV Remote → You press buttons
    • TV System → Works internally
    • You → Only see result

    👉 Remote = API
    👉 You don’t know internal working, only output

    ---

    Key Role of API

    • Provides data
    • Acts as middle layer
    • Connects applications
    API = Communication between systems
    ---

    Calling API in JavaScript

    There are multiple ways:

    • fetch (Most important ✅)
    • axios (Second most important ✅)
    ---

    Fetch API (Core Concept)

    Fetch is a Web API → built into browser
    • No installation required
    • Returns a Promise
    ---

    Basic Syntax

    
fetch("API_URL")
  .then(response => console.log(response))
  .catch(error => console.log(error));
    ---

    Important Concept (VERY IMPORTANT)

    Fetch does NOT return full data directly ❌

    It returns metadata (partial data) first. :contentReference[oaicite:1]{index=1}

    ---

    How to Get Actual Data?

    Use: 

    
response.json()
    ---

    Why .json()?

    • Waits for full data
    • Converts data into JSON format
    • Returns a Promise
    ---

    Final Fetch Flow

    
fetch("API_URL")
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.log(err));
    ---

    Fetch with Async/Await

    
async function getData() {
  const res = await fetch("API_URL");
  const data = await res.json();
  console.log(data);
}
    ---
    Rule:
    • await works ONLY inside async
    • Both fetch & json return Promises
    ---

    Axios (Better Alternative)

    Axios = Third-party library
    • Needs installation (or CDN)
    • Returns Promise
    • Gives full data directly ✅
    ---

    CDN Concept

    CDN = Content Delivery Network

    👉 Access library without installing 👉 Requires internet

    ---

    Axios Syntax

    
axios.get("API_URL")
  .then(res => console.log(res.data))
  .catch(err => console.log(err));
    ---

    Key Difference

    Feature Fetch Axios
    Installation No Yes / CDN
    Data Partial → needs .json() Full data directly
    Ease More steps Simpler
    ---

    What is DOM?

    DOM = Document Object Model

    Breaking it:

    • Document → HTML
    • Object → JavaScript representation
    • Model → Modify/Change
    ---

    Simple Definition

    DOM = Change HTML using JavaScript
    ---

    DOM Flow (Golden Rule)

    
1. Selection
2. Manipulation
    ---

    DOM Selection Methods

    • document.getElementById()
    • document.getElementsByClassName()
    • document.getElementsByTagName()
    • document.querySelector()
    • document.querySelectorAll()
    ---

    Important Behavior

    • ID → Single element
    • Class → Array-like (HTMLCollection)
    • Tag → Array-like
    ---

    Example: Selection + Manipulation

    
let el = document.getElementById("title");
el.style.color = "red";
    ---

    Class Example

    
let items = document.getElementsByClassName("box");

for (let item of items) {
  item.style.color = "blue";
}
    ---

    Query Selector (Best Method)

    
document.querySelector("h1");     // tag
document.querySelector(".class"); // class
document.querySelector("#id");    // id
    ---

    Query Selector All

    
let all = document.querySelectorAll(".box");

all.forEach(item => {
  item.style.color = "green";
});
    ---

    DRY Principle

    DRY = Do Not Repeat Yourself

    Instead of repeating: 

    
el.style.color = "red";
el.style.fontSize = "20px";

    Use: 

    
el.style.cssText = "color:red; font-size:20px;";
    ---

    Final Summary

    
API → Get data
Fetch → Built-in, returns partial data
JSON → Converts to full data
Axios → Easier, returns full data
DOM → Change HTML using JS
