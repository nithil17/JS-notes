    If-Else in JavaScript

    If-else is used to make decisions in JavaScript based on conditions.

    It checks whether a condition is true or false and runs code accordingly.

    Think of it like a real-life situation:
    If it is raining → bring umbrella ☔
    Else → don’t bring umbrella

    Simple If-Else

    If condition is true, code inside if runs. Otherwise, else block runs. 
    
let marks = 55;

if(marks > 75){
  console.log("Graduate");
}else{
  console.log("Not Graduate");
}

    Condition: marks > 75
    55 > 75 → False ❌

    Output: Not Graduate

    If - Else If - Else

    Used when multiple conditions are present. Stops at the first true condition. 
    
let marks = 85;

if(marks > 90){
  console.log("O");
}else if(marks > 80){
  console.log("A");
}else if(marks > 70){
  console.log("B");
}else if(marks > 60){
  console.log("C");
}else{
  console.log("FAIL");
}

    85 > 90 → False
    85 > 80 → True ✅ → Stops here

    Output: A

    Multiple If Statements

    All conditions are checked independently. This can lead to multiple outputs. 
    
let marks = 85;

if(marks > 90){
  console.log("O");
}

if(marks > 80){
  console.log("A");
}

if(marks > 70){
  console.log("B");
}

if(marks > 60){
  console.log("C");
}else{
  console.log("FAIL");
}

    85 > 80 → A
    85 > 70 → B
    85 > 60 → C

    Output: A B C

    Key Difference

    If-Else → One condition
    Else If → Stops at first true condition
    Multiple If → Executes all matching conditions

    Teacher’s takeaway:
    “If you want only one output → use else-if. If you use multiple if → you will get multiple outputs.”
