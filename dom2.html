    DOM Manipulation & Events (Teacher Style Explanation)

    In this session, we move deeper into DOM manipulation and understand how JavaScript interacts with HTML elements dynamically.

    ---

    1. DOM Basics Recap

    Two Core Steps:
    • Selection → Select the element
    • Manipulation → Modify the element

    Selection Methods:

    
document.getElementById()
document.getElementsByClassName()
document.getElementsByTagName()
document.querySelector()
document.querySelectorAll()
    ---

    2. Getting Text from Elements

    Methods:

    • innerText
    • textContent
    • innerHTML
    ---

    Example:

    
let h1 = document.querySelector("h1");

console.log(h1.innerText);
console.log(h1.textContent);
console.log(h1.innerHTML);
    ---

    Key Differences (VERY IMPORTANT)

    Teacher Insight:
    • innerText → "Smart" → respects CSS (hidden text not shown)
    • textContent → "Brainless" → shows everything (even hidden)
    • innerHTML → returns text + HTML tags
    ---

    Example with Hidden Content

    
<h1>Hello <span style="display:none">Hidden</span></h1>

    
h1.innerText      // "Hello"
h1.textContent    // "Hello Hidden"
    ---

    Getter & Setter Behavior

    
h1.innerText = "New Text";

    👉 These methods can both get and set values

    ---

    3. innerHTML Power

    
h1.innerHTML = "<em>Hello</em>";

    Output: Hello (styled as italic)

    👉 innerHTML allows injecting full HTML elements

    ---

    4. Working with Attributes

    Getter:

    
element.getAttribute("type");

    Setter:

    
element.setAttribute("type", "password");
    ---

    Example:

    
let input = document.querySelector("input");

input.getAttribute("type");   // "text"
input.setAttribute("type", "password");
    ---

    Anchor Example

    
anchor.setAttribute("href", "https://google.com");
anchor.setAttribute("target", "_blank");
    ---

    5. classList (Better Way)

    Instead of using setAttribute for class, use classList
    ---

    Methods:

    
element.classList.add("className");
element.classList.remove("className");
element.classList.toggle("className");
element.classList.contains("className");
    ---

    Example:

    
h1.classList.add("red");
h1.classList.remove("red");
    ---

    Toggle Concept (IMPORTANT)

    "Toggle means reverse the state" 

    
h1.classList.toggle("active");
    • If class exists → removed
    • If class not exists → added
    ---

    Teacher Analogy

    Toggle = Fan Switch ON → OFF OFF → ON

    ---

    6. DOM Tree Traversal

    Moving inside DOM without selecting again

    ---

    Navigation Methods:

    • nextElementSibling
    • previousElementSibling
    • parentElement
    • children
    ---

    Example:

    
let item = document.querySelector("li");

item.nextElementSibling.style.color = "green";
item.previousElementSibling.style.color = "orange";
    ---

    Parent Access

    
item.parentElement.style.border = "2px solid black";
    ---

    Children Access

    
item.parentElement.children[0].style.background = "yellow";

    👉 children returns array-like structure

    ---

    7. Creating Elements Dynamically

    Steps:

    1. Select parent
    2. Create element
    3. Add content
    4. Append to DOM
    ---

    Example:

    
let div = document.querySelector("div");

let p = document.createElement("p");
p.innerText = "Hello from JS";

div.appendChild(p);
    ---

    Append vs Prepend

    
div.append(p);   // end
div.prepend(p);  // start
    ---

    8. Events in JavaScript

    Definition:
    Event = User action (click, type, scroll, etc.)
    ---

    Examples of Events:

    • click
    • dblclick
    • keydown
    • keyup
    • input
    ---

    9. Ways to Handle Events

    1. Inline Method (Not Recommended)

    
<button onclick="handleClick()">Click</button>
    ---

    2. JavaScript Property

    
button.onclick = handleClick;

    ❌ Problem: Only one function allowed (overrides)

    ---

    3. addEventListener (BEST)

    
button.addEventListener("click", handleClick1);
button.addEventListener("click", handleClick2);

    ✅ Multiple functions can run

    ---

    10. Input Events

    Key Events:

    • keydown → when key pressed
    • keyup → when key released
    • input → when value changes
    ---

    Example:

    
input.addEventListener("keydown", (event) => {
  console.log(event.target.value);
});
    ---

    Important Rule

    For input fields → use value NOT innerText / textContent
    ---

    Event Object

    
input.addEventListener("keydown", (event) => {
  console.log(event);
});

    👉 Event contains full info about action :contentReference[oaicite:0]{index=0}

    ---

    11. Image Carousel Logic (Very Important)

    Goal:

    Change image every 2 seconds and loop back

    ---

    Code:

    
let images = ["img1.jpg", "img2.jpg", "img3.jpg"];

let i = 0;

setInterval(() => {
  img.src = images[i];
  i = (i + 1) % images.length;
}, 2000);
    ---

    Concept:

    • Modulo (%) keeps index in range
    • Creates infinite loop cycle
    ---

    Teacher Insight (Very Important)

    "DOM is just a small part — rest is pure JavaScript logic"
    ---

    Final Summary

    
Selection → querySelector
Manipulation → innerText, innerHTML
Attributes → getAttribute / setAttribute
Classes → classList
Traversal → parent / children / siblings
Events → addEventListener
Input → use value
Dynamic → createElement + append
